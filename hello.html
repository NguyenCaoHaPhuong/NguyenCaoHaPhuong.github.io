TU LUYỆN ĐẤU KHÍ
Đấu Khí (Nhập môn): Nhất Đoạn -> Cửu Đoạn
Đấu Giả: Nhất Tinh -> Cửu Tinh
Đấu Sư: Nhất Tinh -> Cửu Tinh
Đại Đấu Sư: Nhất Tinh -> Cửu Tinh
Đấu Linh: Nhất Tinh -> Cửu Tinh
Đấu Vương: Nhất Tinh -> Cửu Tinh
Đấu Hoàng: Nhất Tinh -> Cửu Tinh
Đấu Tông: Nhất Tinh -> Cửu Tinh
Đấu Tôn: + Sơ Giai: Nhât Tinh -> Tam Tinh + Trung Giai: Tứ Tinh -> Lục Tinh + Cao Giai: Thất Tinh -> Cửu Tinh Đỉnh Phong + Đỉnh Phong: Nhất Chuyển -> Cửu Chuyển
Bán Thánh: Sơ Cấp -> Trung Cấp -> Cao Cấp
Đấu Thánh (Mỗi Tinh phân chia Sơ Kỳ -> Trung Kỳ -> Hậu Kỳ -> Đỉnh Phong): Từ Ngũ Tinh trở lên có sự thay đổi vể chất, Cửu Tinh Đấu Thánh có thể làm cho linh hồn lực đạt Viên Mãn (Đế Cảnh) + Tiểu Thành: Nhất Tinh -> Tứ Tinh + Đại Thành: Ngũ Tinh -> Bát Tinh + Đại Viên Mãn: Cửu Tinh -> Cửu Tinh Đỉnh Phong + Bán Đế
Đấu Đế



Đấu Đế (Địa Chí Tôn): Hạ Phẩm -> Thượng Phẩm -> Đại Viên Mãn -> Chuẩn Thiên Chí Tôn (Địa Chí Tôn Đỉnh Phong)
Thiên Chí Tôn:
+ Linh Phẩm: Sơ Kỳ -> Trung Kỳ -> Hậu Kỳ -> Đỉnh Phong -> Bán Tiên Phẩm
+ Tiên Phẩm: Sơ Kỳ -> Trung Kỳ -> Hậu Kỳ -> Đỉnh Phong -> Bán Thánh Phẩm
+ Thánh Phẩm: Sơ Kỳ -> Trung Kỳ -> Hậu Kỳ -> Đỉnh Phong -> Siêu Việt
